# Проект: Вычисление геометрических характеристик прямоугольника

## 1. Описание проекта
Данный проект представляет собой консольное приложение на языке C++, предназначенное для вычисления основных геометрических параметров прямоугольника:
- **Периметр**;
- **Площадь**;
- **Длина диагонали**.

Проект разработан в рамках изучения основ программирования, стандартной библиотеки C++, а также получения практических навыков работы с системой контроля версий **Git** и платформой **GitHub**.

## 2. Цель и задачи проекта
**Цель:** Создание модульной программы с разделением кода на интерфейс, логику и тестирование.

**Задачи:**
1.  Реализовать функции для вычисления периметра, площади и диагонали.
2.  Вынести объявления функций в заголовочный файл (`GorinaLida.h`).
3.  Реализовать отдельный модуль тестирования (`test.cpp`) с использованием механизма `assert`.
4.  Освоить базовые команды Git (commit, push, pull) для отслеживания изменений.
5.  Разместить репозиторий на GitHub и оформить документацию.

## 3. Состав проекта (Структура файлов)
Проект организован в соответствии с принципом разделения кода на интерфейс, реализацию и тестирование:

- GorinaLida.h # Заголовочный файл (объявление функций)
- GorinaLida.cpp # Основной модуль (реализация функций)
- test.cpp # Модуль тестирования (unit-тесты)
- README.md # Документация проекта

**GorinaLida.cpp** (Реализация)
содержит:
- Периметр - Perimetr ((l+w)*2)
- Площадь - Area (l*w)
- Диагональ - Diagonal ((l^2+w^2)^0.5) по теореме Пифагора

**test.cpp** (Тестирование)
Содержит набор проверок с использованием assert. Проверяет корректность работы функций на заранее известных значениях (прямоугольник 3*4).
